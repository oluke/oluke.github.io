Research (Data Science & Machine Learning)

  • Fine-tuning Machine Learning Models for DNA Sequence Analysis with K-mer Size Dec 2023

    Four ML models (KNN, NB, RF, and SVM) were analyzed for k-mer sizes and hyperparameter optimization. SVM achieved the highest accuracy of 91.8% at a 5-mer size, while NB reached 88.2% at an 8-mer size.

  • Comparison of Edge Detection Methods for Image Extraction Dec 2022

    The Laplacian algorithm performed the best, extracting contours with 89.88% accuracy, without any noise or broken lines.

  • Genus Classification using Convolution Neural Network (CNN) Aug 2022

    CNN was used to classify images of three fish genera: Epinephelus spp., Halichoeres spp., and Lutjanus spp. The model achieved an accuracy of 85.31%, with precision at 89.92% and sensitivity at 86.49%.

  • Data Scaling Analysis on Machine Learning Models Jan 2022

    Data scaling methods (min-max and zero-mean) were tested with ML models. SVM performed better with scaling, while Random Forest performed best without it, achieving 74.87% accuracy.

  • Mangrove Identification Using Machine Learning Dec 2021

    Data preprocessing and min-max normalization were applied to improve accuracy. Random Forest achieved 99.97% accuracy on average, with a maximum of 100% in some configurations to identify six mangrove species.
